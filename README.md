# My Practice AI Projects

A collection of AI-powered web tools and utility apps built with HTML, CSS, and JavaScript. Most projects use the Claude API (Anthropic) to generate content, analyze data, and automate tasks.

## Projects

### AI-Powered Tools

| Project | Description |
|---|---|
| [Batch Product Generator](batch-product-generator.html) | Generate product descriptions at scale — up to 50 products with brand voice selection (Luxury, Professional, Friendly, etc.), real-time progress tracking, and CSV export |
| [CRM Formatter](crm-formatter.html) | Paste unstructured contact data and auto-parse it into structured fields with CSV/JSON export |
| [Email Triage](email-triage.html) | Paste a batch of emails and automatically categorize them (Support, Billing, Spam, etc.) with priority levels and response starters |
| [Glowbox Support](glowbox-support.html) | Generate customer support replies for a skincare subscription service — detects issue type and adjusts tone |
| [Grant Writer](grant-writer.html) | AI-assisted grant proposal and funding application writer with section-by-section composition |
| [Listing Generator](listing-generator.html) | Input property details and generate SEO-optimized real estate or rental listing copy |
| [Meeting Summarizer](meeting-summarizer.html) | Paste meeting notes or transcripts and extract action items, decisions, and key takeaways |
| [Menu Generator](menu-generator.html) | Generate restaurant menus with descriptions from ingredients or cuisine type |
| [Retail Dashboard](retail-dashboard.html) | Sales analytics and inventory dashboard with AI-generated insights |
| [SEO Blog Generator](seo-blog-generator.html) | Generate SEO-optimized blog posts with keyword integration, tone selection, and word count control |
| [Social Calendar Generator](social-calendar-generator.html) | Generate a social media content calendar with captions and hashtags across platforms |
| [Vendor Onboarding](vendor-onboarding.html) | Multi-step vendor onboarding workflow with AI-assisted data processing and document analysis |

## Usage

Each project is a self-contained HTML file. Open any file directly in a browser to use it.

Projects that use the Claude API will prompt you to enter your Anthropic API key. You can get one at [console.anthropic.com](https://console.anthropic.com).

## Tech Stack

- HTML / CSS / JavaScript (vanilla, no frameworks)
- Claude API (Anthropic) — `claude-opus-4-6`
